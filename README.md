# maos-que-ensinam

# Descrição:

O app é direcionado a estudantes de 14 a 18 anos (ensino médio) e funciona por meio de um sistema de tutoria entre pares. 

    Ele conecta:

•Alunos tutores: Estudantes com bom desempenho acadêmico.
•Alunos tutorados: Estudantes neurodivergentes ou que enfrentam dificuldades de aprendizagem.

    Desafios que Visa Resolver:

•Evasão escolar.
•Desigualdade no aprendizado.
•Suporte pedagógico insuficiente.

    Funcionalidades e Diferenciais:

•Plataforma Gamificada: Utiliza dinâmicas de jogos para engajar os alunos.
•Trilhas de Capacitação: Oferece módulos específicos (como introdução ao TEA - Transtorno do Espectro Autista, TDAH e fundamentos da educação inclusiva) para preparar os tutores.
•Conexão com Instituições: O aplicativo integra as escolas na plataforma para acompanhar o progresso e gerar impacto social.

Em suma, é uma solução digital de alto impacto e escalabilidade que busca promover a inclusão escolar de forma acessível, garantindo que estudantes neurodivergentes tenham as mesmas oportunidades de desenvolvimento.
